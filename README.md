## Описание

- К сожалению, данный проект был отменен на этапе первого MVP, поэтому протестировать его не получится из-за того, что API отключено. Но вы можете ознакомиться с качеством написанного мною кода для клиентской части.
- Данный проект задумывался как мощное отечественное приложение для комфортного изучения иностранных языков.
- Специально для этого приложения я создал дизайн систему с нуля без использования сторонних UI китов.
- Был реализован функционал для работы со словарем (добавление, удаление слов, редактирование, сортировка и пагинация), функционал для тренировок выученных слов, приветственная страница и страница с настройками пользователя.
- Внешний вид страницы соответствует макету и визуально не отличается на современных десктопных и мобильных браузерах (за исключением числа колонок в строке).
- Код соответствует стандартам ES2015+. В приложении широко применяются принципы DRY, KISS, YAGNI и SOLID.

## Технологический стек

При разработке проекта использовались следующие технологии:

- **TypeScript и React** - для разработки;
- **Next.js** - для реализации SSR;
- **Redux Toolkit** - в качестве стейтменеджера и инструмента для интеграции с API;
- **TailWind** - для стилизации;
- **ESLint, Prettier и Husky** - для линтинга и форматирования.
